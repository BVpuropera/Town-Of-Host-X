・ピエロ

 クルーだがシェイプシフトすることができる。
　人外をキルすることもできる。クルーメイトの場合キルがキャンセルされる。
・ベントスター

　ベントを使用できるスター。タスクはない。
　アンチにはキルされない。最初から名前が光ってる。
　ベントのクールダウンは0秒。

コンビネーション
・スターとアンチ
（アンチ）
陣営：第三陣営　判定：インポスター　置き換え役職：インポスター
スター以外をキルしようとすると死ぬニュートラル。
スターをキルした瞬間アンチの単独勝利で試合を終わらせる。
（スター）
陣営：クルー　判定：インポスター　置き換え役職：インポスター
アンチが死んだ場合名前が光るようになるクルーメイト。
タスクはない。キルボタンを押すとキルターゲットを強制的に役職をクルーにしてキルをキャンセルする。
キルターゲットの陣営がクルーだった場合役職はクルーのままにする（インポスター役職「記憶喪失者」の場合役職はシェリフになる）
一度キルをキャンセルすると役職がベントスターになる。ベントスターになった場合タスクが追加される。
・キラとL
（キラ）
陣営：第三陣営　判定：シェイプシフター　置き換え役職：シェイプシフター
シェイプシフトした相手を殺す。
もしシェイプシフトした相手がLだった場合キラの単独勝利。
（L）
陣営：第三陣営　判定：インポスター　置き換え役職：インポスター
キルをするとキルターゲットがキルターゲットの死体をすぐに通報する。
その後システムメッセージでキルターゲットの陣営とキルターゲットの役職を公表する。
もし逮捕した対象がキラだった場合、Lとクルーの勝利とする。
もしタスクでクルーが勝ってしまうとLは敗北する。
